# customer-behavior-analysis
Customer Behavior Analysis & Prediction using Transaction Data

#📌 Overview
This project focuses on analyzing customer transaction data to understand purchasing behavior and predict future actions such as churn and repeat purchases. It helps businesses make data-driven decisions to improve customer retention and revenue.

#🎯 Objectives
-Identify customers likely to churn
-Predict repeat purchase probability
-Segment customers using RFM analysis
-Provide actionable insights for business strategy

#🧠 Key Features
-RFM (Recency, Frequency, Monetary) feature engineering
-Machine Learning models:
            -Random Forest → Churn Prediction
            -Logistic Regression → Repeat Purchase Prediction
            -K-Means → Customer Segmentation
-Interactive prediction interface using Gradio

#⚙️ Tech Stack
-Python (Pandas, NumPy, Scikit-learn)
-Machine Learning
-Gradio (UI)
-Git & GitHub

#📂 Project Structure
-app/        → Gradio app  
-src/        → Data processing and business logic  
-models/     → Trained ML models  
-data/       → Sample datasets  
-notebooks/   → Exploratory analysis  

#▶️ How to Run
-pip install -r requirements.txt
-python app/gradio_app.py

#📊 Business Impact
-Helps identify at-risk customers
-Enables targeted marketing campaigns
-Improves customer retention strategies
-Supports data-driven decision-making
